• 显示已连接，却打不开Google或YouTube

• 用着用着突然断线

• 速度慢到像“回到拨号时代”

• 频繁DNS错误

再加上很多免费服务并没有针对国内环境做优化，节点远、线路绕、没有混淆能力，体验自然很难稳定。
